# Hackysack World

A modern website dedicated to the sport of hackysack (footbag). This site provides comprehensive information about hackysack, training techniques, and instructional videos.

## Features

- Learn about the history and rules of hackysack
- Explore different hackysack moves and tricks
- Follow step-by-step training techniques
- Watch instructional videos
- Community forum for hackysack enthusiasts

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.
